## A* Crusade

  Space Crusade (tm) HTML Canvas Clone.

### Status

  * pending 2019-05-21

### File System

  * doc -> documents (txt, odg)
  * lib -> libraries (js)
  * res -> resources (css, woff, png)
  * src -> sources (js)
